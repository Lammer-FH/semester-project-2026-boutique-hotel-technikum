# Boutique Hotel Technikum

Full-stack booking application for the `Boutique Hotel Technikum` for FHTW course 'Advanced Webtechnologies'.

## Pre-requisites

- Backend: Java 21, Spring Boot 3.4.x
- Frontend: Ionic 8.x, Vue.js 3.4.x, Pinia 2.1.x
- Database: MySQL 8.4 LTS

## Quick Install

1. Install `Java 21` `Node.js` and `MySQL 8.4`
2. Set up the backend and frontend projects in their respective folders.
